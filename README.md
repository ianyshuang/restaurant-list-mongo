# 我的餐廳清單
整合式的餐廳清單，使用者可操作基礎的CRUD（新增 / 讀取 / 更新 / 刪除）功能

## Features 
- 可新增、讀取、更新、刪除餐廳資料
- 可針對所有餐廳種類進行篩選(Filter)
- 可按照（中文名稱 / 英文名稱 / 評分高低）進行餐廳排序
- 可對餐廳名稱及種類進行搜尋
- 可在每一間餐廳中新增評論至評論區，系統將自動生成評論日期
- 刪除前會提醒使用者是否要刪除以免誤按
